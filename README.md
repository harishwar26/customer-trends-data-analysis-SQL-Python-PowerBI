# customer-trends-data-analysis-SQL-Python-PowerBI

## 📌 Project Overview

This project focuses on analyzing customer purchasing behaviors and trends to provide actionable business insights. By leveraging **SQL** for data management, **Python** for deep exploratory analysis, and **Power BI** for interactive storytelling, I identified key drivers behind customer retention and sales growth.

## 🛠️ Tech Stack

* **Database:** SQL (Data Extraction & Cleaning)
* **Language:** Python (Pandas, Matplotlib, Seaborn)
* **Visualization:** Power BI (Interactive Dashboards)

## 🚀 Key Workflow

1. **Data Extraction (SQL):** Cleaned raw transactional data and joined multiple tables (Customers, Products, Sales) to create a unified dataset.
2. **Exploratory Data Analysis (Python):** Used Python to identify correlations, handle missing values, and perform statistical analysis on customer demographics.
3. **Insight Visualization (Power BI):** Developed a dynamic dashboard to track KPIs such as Monthly Revenue, Customer Churn Rate, and Top-Performing Categories.
